Bem-vindo a API de Produtos.

Este projeto foi desenvolvido ao longo das aulas de Modelagem de Banco de dados da Pós em Desenvolvimento Web Full Stack da UNIPÊ. O projeto lida com uma persistência poliglota, com dados em um banco relacional e um não relacional.

Relacional MySQL com as tabelas:

  1. Produtos (que mantém): ID_produto, Nome, Descricao, Categoria e Preço
  2. Clientes: ID_cliente, Nome, Email, CPF e Data de Nascimento

Não reacional MongoDB com a coleção:

  1. Pedidos: ID_produto, ID_Cliente, Data_pedido, Valor_pedido

O projeto foi implementado em Python com uso do Flask. As dependências e pacotes utilizados estão no arquivo requirements.txt
